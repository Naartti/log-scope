# Usage
## Installation
```npm install scope-log```

## Get levels from IPs (IPv4)
```js
import scopeLog from 'scope-log'
const log = scopeLog('scopeName')

log.todo('Remember this')
```

# Developement
- ```npm install```
- ```npm run dev```
- Go to localhost:8080/scope-log

# Production
Run lint, tests and builds for production
```bash
npm run prod
```
